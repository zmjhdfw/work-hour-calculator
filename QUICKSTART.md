# 快速入门指南

## 5分钟快速上手

### 1. 运行程序

```bash
python run.py
```

### 2. 添加第一条工时记录

1. 选择菜单 `1. 添加工时记录`
2. 输入日期（直接回车使用今天）
3. 输入开始时间：`09:00`
4. 输入结束时间：`17:30`
5. 输入项目名称：`我的项目`
6. 输入描述（可选）：`开发新功能`

### 3. 查看记录

选择菜单 `2. 查看工时记录` → `1. 查看所有记录`

### 4. 查看统计

选择菜单 `5. 计算工时统计` → `4. 计算所有记录`

### 5. 生成报告

选择菜单 `6. 生成统计报告` → `1. 生成摘要报告`

### 6. 导出数据

选择菜单 `7. 导出数据` → `1. 导出所有记录`
输入文件名：`my_work_hours.csv`

## 常用操作

### 按日期查看

菜单 `2` → `2. 按日期查看` → 输入日期

### 按项目查看

菜单 `2` → `4. 按项目查看` → 选择项目

### 编辑记录

菜单 `3` → 输入记录ID → 修改字段（留空保持原值）

### 删除记录

菜单 `4` → 输入记录ID → 确认删除

## 数据格式

- **日期**: YYYY-MM-DD (例如: 2026-04-10)
- **时间**: HH:MM (例如: 09:00)
- **标准工时**: 8小时/天（可在config.py修改）

## 提示

- 数据自动保存在 `work_records.json` 文件中
- 超过8小时的工时会被标记为加班
- 可以按项目统计工时分布
- 支持导出CSV格式，方便用Excel打开

## 示例工作流程

```
1. 每天开始工作：添加记录（开始时间）
2. 每天结束工作：添加记录（结束时间）或编辑记录
3. 每周查看：生成统计报告
4. 月底导出：导出CSV文件用于汇报
```
